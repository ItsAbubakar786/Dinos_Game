# 🦖 Dino Game

A simple and fun browser-based Dino Game built using HTML, CSS, and JavaScript.  
Inspired by the classic Chrome offline dinosaur game.

---

## 🎮 Features

- 🕹️ Jump over obstacles
- ⚡ Increasing difficulty with time
- 🎯 Score tracking system
- 💥 Game over detection
- 🔄 Restart functionality
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---
---

## ▶️ How to Play

1. Open the game in your browser  
2. Press **Spacebar** or **Arrow Up** to jump  
3. Avoid obstacles  
4. Try to score as high as possible
